This repository realizes them architecturally - `tmf.model.mixin` (`tmf_base`), JSON characteristic fields, and the generic `tmf.entity` models - rather than one Odoo model per class, so entities are not enumerated here. Full entity detail is available in the JSON export via `--include-patterns-entities`.
